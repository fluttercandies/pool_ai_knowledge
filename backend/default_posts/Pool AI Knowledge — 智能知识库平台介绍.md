## 什么是 Pool AI Knowledge？

Pool AI Knowledge 是一个基于 **RAG（检索增强生成）** 技术的智能知识库平台。它将传统的内容管理与前沿的 AI 技术相结合，让您的知识资产真正"活"起来。

### 核心能力

- **智能语义搜索**：基于 OpenAI Embeddings + FAISS 向量检索，理解用户意图而非简单关键词匹配
- **AI 对话问答**：集成 Google Gemini 模型，用户可以用自然语言与知识库对话
- **多语言支持**：支持中文、英文、日文、韩文等多种语言的内容管理与检索
- **Markdown 编辑**：文章内容完全支持 Markdown 格式，排版自由灵活

### 技术架构

```
前端 (Vue 3 + Element Plus)
    ↕
后端 (FastAPI + Python)
    ↕
┌─────────────────────────────────┐
│  MySQL — 结构化数据存储          │
│  FAISS — 向量索引（语义检索）     │
│  OpenAI — 文本嵌入向量生成       │
│  Google Gemini — AI 对话生成     │
└─────────────────────────────────┘
```

### 适用场景

1. **企业内部知识库**：沉淀团队经验，新人快速上手
2. **产品文档中心**：用户可通过 AI 对话快速找到答案
3. **技术博客平台**：Markdown 写作 + 智能检索
4. **客服辅助系统**：AI 自动从知识库中检索答案并回复

> Pool AI Knowledge — 让知识流动，让 AI 赋能。
