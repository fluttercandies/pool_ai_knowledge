## 2025年3月 — 前端开发与部署

### 3月3日 — Vue 3 前端项目初始化

用 Vite 创建了 Vue 3 前端项目，引入了 Element Plus 组件库和 vue-i18n 国际化。

页面规划了四个主要路由：
- `/` — 首页
- `/posts` — 文章列表
- `/posts/:id` — 文章详情（Markdown 渲染）
- `/chat` — AI 对话页面

Vite 的开发体验真的很好，热更新几乎是瞬间的。

### 3月10日 — 文章列表和搜索页面

完成了文章列表页面，支持分页、按语言筛选。搜索功能对接了后端的 RAG 语义搜索接口。

搜索结果展示做了高亮处理，用户体验比较好。文章详情页用了 markdown-it 渲染，代码块支持语法高亮。

### 3月17日 — AI 对话界面

做了一个类 ChatGPT 的对话界面：
- 消息气泡样式区分用户和 AI
- AI 回复支持 Markdown 渲染
- 底部显示引用的文章来源，点击可跳转
- 支持切换对话语言

遇到的问题：AI 响应时间有时候比较长（3-5秒），加了 loading 动画和"AI 正在思考"提示。

### 3月24日 — 管理后台对接

管理后台用的是 vue-element-admin 模板（Vue 2），已经有了完整的登录、路由守卫、权限管理。

对接了后端的管理 API：
- 文章管理（增删改查 + Markdown 编辑器）
- API 密钥管理（添加/删除 OpenAI 和 Google 密钥）
- 模型切换（可选 Gemini 不同版本）
- 管理员账户管理

### 3月31日 — Linux 部署脚本

写了一键部署脚本 `deploy.sh`，支持 Ubuntu/Debian/CentOS：
- 自动安装 Python、Node.js、MySQL、Nginx
- 构建前端并部署到 Nginx
- 配置 systemd 服务自启动
- 生成 `.env` 配置文件

在阿里云 2C4G 的 ECS 上测试部署，全程大约 5 分钟。
